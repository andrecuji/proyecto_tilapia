HERRAMIENTAS UTILIZADAS

ESP32: MICROCONTROLADOR

Descripcion: Un microcontrolador de bajo costo y alto rendimiento, que incluye conectividad Wi-Fi y Bluetooth. Es ampliamente utilizado en proyectos de IoT (Internet de las Cosas) y es compatible con el entorno de desarrollo de Arduino.

Aplicacion: Se utilizo para controlar sensores y dispositivos, lo que nos permite realizar un monitoreo remoto de los diferentes datos obtenidos.

PH45: SENSOR DE PH

Descripcion:Un sensor diseñado para medir el nivel de pH de líquidos, útil para evaluar la acidez o alcalinidad de una solución. Este tipo de sensor es común en proyectos que requieren monitoreo de calidad del agua.

Aplicacion: Se utilizó para monitorear la calidad del agua en un estanque de tilapia

DS18B20 SUMERGIBLE: SENSOR DE TEMPERATURA

Descripcion:Un sensor de temperatura digital con encapsulamiento impermeable, ideal para medir temperaturas en entornos húmedos o directamente en líquidos. Tiene una alta precisión y permite medir desde -55°C a +125°C.

Aplicacion: Nos ayuda a monitoriar la temperatura de lso estanques, para poder tener un control ambiental.

PROTOBOARD

Descripcion: También conocida como placa de pruebas, es una herramienta utilizada para construir circuitos electrónicos de forma temporal sin necesidad de soldar. Permite conectar componentes como resistencias, LEDs y microcontroladores, facilitando la realización de prototipos y experimentos.

RESISTENCIA

Descripcion: Es un componente electrónico que limita la cantidad de corriente que fluye a través de un circuito. Se mide en ohmios (Ω) y se utiliza para proteger otros componentes, dividir voltajes o establecer corrientes en circuitos, asegurando un funcionamiento adecuado y seguro.
